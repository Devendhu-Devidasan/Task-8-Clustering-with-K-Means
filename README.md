# Task-8-Clustering-with-K-Means
AI &; ML Internship - task  8Task 8: Clustering with K-Means

>Objective



To perform unsupervised learning using K-Means clustering algorithm.

> About the Project


In this project, I generated a synthetic dataset using sklearn's make_blobs function and applied K-Means clustering to group the data points into clusters.



> Tools & Libraries Used



- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

>🔍 Steps Performed


1. Imported required libraries
2. Generated synthetic dataset using make_blobs
3. Visualized the dataset
4. Used Elbow Method to find optimal number of clusters
5. Applied K-Means clustering
6. Visualized clustered data
7. Evaluated model using Silhouette Score
